### 学習内容まとめ

#### 1. OpenAI API呼び出しの章 → `core/openai_adapter.py`
- `openai.ChatCompletion.create()` でAPI呼び出し
- クラス設計：コンストラクタでsystem_promptを自動設定

#### 2. VOICEVOX連携の章 → `core/voicevox_adapter.py`
- `get_voice(text)` にテキストを渡すと `(data, rate)` の2つの情報を返す
- 2段POST方式：`/audio_query` → `/synthesis`（96頁詳細）

#### 3. 音声再生の章 → `core/play_sound.py`
- バイト列を音声データに変換 → sounddeviceで再生
- PlaySoundクラスの要件：「指定したサウンドデバイスに音を出力する」（92頁詳細）
- `sd.play()` + `sd.wait()` で同期再生

#### 4. メインループ・統合の章 → `core/aituber_system.py` → `core/run_dummy.py`
- `aituber_system.py`：各モジュールを呼び出し、コメント取得→応答生成→音声出力を一連処理（139頁詳細）
- `run_dummy.py`：`talk_with_comment` を5秒ごとに繰り返し実行するスクリプト
