### 新規ファイル詳細

#### `memory/memory_agent.py`
- `GraphDatabase.driver` で Neo4j に接続（.envのURI/USER/PASSWORD参照）
- `save_interaction()`: MERGE でViewerノード作成/更新 + Commentノード作成 + COMMENTEDリレーション
- `get_memory()`: 過去3件のコメントを新しい順に取得、JST変換して文字列で返す
- 例外処理: Neo4j接続エラー時はログ出力して空文字列を返す（システム停止なし）

#### Neo4jデータモデル
```
(:Viewer {name, first_seen, last_seen})-[:COMMENTED]->(:Comment {text, timestamp, emotion, response})
```
