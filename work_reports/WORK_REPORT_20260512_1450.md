## 発生したエラーと対処
| エラー内容 | 原因 | 対処法 |
|-----------|------|------|
| `TypeError: argument of type 'NoneType' is not iterable` (pytchat) | `YOUTUBE_VIDEO_ID` が未設定で `YoutubeCommentAdapter(video_id=None)` を呼び出した | `video_id` が `None` の場合は初期化をスキップする条件分岐を追加 |
| `Exception: OBSの設定がされていません` | OBS未起動時に `OBSAdapter()` が例外を投げた | `try-except` で `obs_adapter=None` とし、使用箇所に `None` チェックを追加 |
| `ModuleNotFoundError: No module named 'google_auth_oauthlib'` | YouTube APIパッケージが未インストール | `pip install google-auth-oauthlib==1.2.1 google-api-python-client==2.147.0 google-auth-httplib2==0.2.0 pyyaml` で個別インストール |
