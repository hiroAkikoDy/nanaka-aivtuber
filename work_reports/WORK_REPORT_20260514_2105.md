```python
    output_device_id = self._search_output_device_id(output_device_name)

def __init__(self, output_device_name="CABLE Input"):
    self.available = False
    if output_device_name is None:
        return  # None時はスキップ
    output_device_id = self._search_output_device_id(output_device_name)
    if output_device_id is None:
        return  # デバイス未検出時はexit()せずスキップ
    sd.default.device = [None, output_device_id]
    sd.default.channels = [None, 2]
    self.available = True
```

## 発生したエラーと対処
| エラー内容 | 原因 | 対処法 |
|-----------|------|--------|
| `TypeError: 'in <string>' requires string as left operand, not NoneType` | settings.yamlのoutput_device_nameをnullに変更 | play_sound.pyでNone判定を追加 |
| `PlaySound出力デバイス 'CABLE Input' が見つかりません` | VB-Cableドライバが認識されていない | settings.yamlを"Realtek"に変更 |
| `PlaySound出力デバイス 'Speakers' が見つかりません` | デバイス名が"スピーカー"（日本語） | "Realtek"でマッチ |
| `Error opening OutputStream: Invalid number of channels [-9998]` | index=1（マイク）が出力デバイスとして選択された | max_output_channels > 0でフィルタ |
| VOICEVOX応答中のKeyboardInterrupt | ユーザーがCtrl+Cで中断 | バグではない（正常動作） |
