TypeScriptが使われる理由
核心：AIVTuberは「フロントエンドアプリケーション」だから
AIVTuberに必要なもの：
  ① キャラクターの描画（Live2D・VRM・表情変化）
  ② リアルタイム音声（口パク同期・感情表現）
  ③ 配信ソフト連携（OBS・VTube Studio）
  ④ ブラウザ・デスクトップアプリとして動く

これらは全てJavaScript/TypeScriptの領域
具体的に見ると：
① キャラクター描画
Three.js  → VRMモデルの3D描画
PixiJS    → Live2Dモデルの2D描画
@pixiv/three-vrm → VRM規格の処理

→ ブラウザで動くのはJavaScriptのみ
  Pythonでブラウザレンダリングはできない
② リアルタイム音声・口パク
Web Audio API  → ブラウザ標準の音声処理
WebSocket      → 音声とアニメーションのリアルタイム同期

→ ブラウザのAPIはJavaScriptからしか叩けない
③ デスクトップアプリ化
Project AIRI → Tauri（Rust + TypeScriptフロントエンド）
AITuberKit  → Electron（Node.js + TypeScript）

→ ウィンドウを持つデスクトップアプリを
  Web技術で作るフレームワークはTypeScript前提

PythonとTypeScriptの役割分担

【TypeScript層（フロントエンド）】

  ↕ API通信
【Python層（バックエンド）※必要に応じて】
  LLM呼び出し・RAG・記憶管理・データ処理

Project AIRIはTypeScriptでLLMのAPI呼び出しも
